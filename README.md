# restaurant-page 

## Live Demo: https://sophlanz.github.io/restaurant-page/

## Description 
A simple mockup website for a restaurant. 

## Tools and Technologies
- Javascript 
- HTML5
- CSS3

## Features
- Pop-up reservation window.
- Use of tabbing for the navigation bar, and various menus.
